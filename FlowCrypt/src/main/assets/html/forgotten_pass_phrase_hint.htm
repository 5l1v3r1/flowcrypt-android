Forgotten pass phrase?

If you are setting up FlowCrypt on a second device and your first device can still read encrypted email, you can change your pass phrase on the original device before continuing:

  1. Open "FlowCrypt Settings (on that device) -> Security -> Change pass phrase" on that original device
  2. Set a new pass phrase
  3. Restart FlowCrypt Android App
  4. Use your new pass phrase to set up the Android App
